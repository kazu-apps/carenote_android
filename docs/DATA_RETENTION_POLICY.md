# データ保持ポリシー

CareNote（ケアのて）は、個人情報保護法（APPI）に基づき、
利用者の個人情報を適切に取り扱います。

## 個人情報の定義

本アプリが取り扱う個人情報（PII）には以下が含まれます:

| カテゴリ | 具体例 |
|---------|--------|
| 認証情報 | メールアドレス、パスワード（ハッシュ化）、表示名 |
| ケア対象者情報 | 氏名、ニックネーム、生年月日、性別、要介護度 |
| 健康情報（要配慮個人情報） | 体温、血圧、脈拍、体重、食事量、排泄状況、体調メモ |
| 服薬情報 | 薬剤名、用量、服用タイミング、服薬記録 |
| 医療関連情報 | 既往歴、アレルギー情報 |
| 連絡先情報 | 緊急連絡先（氏名、電話番号、続柄） |
| 写真データ | 健康記録・メモに添付された写真 |

## 利用目的

収集した個人情報は、以下の目的にのみ使用します:

1. **介護記録の管理**: 服薬、健康状態、タスク、メモの記録・表示
2. **通知機能**: 服薬リマインダー、タスクリマインダーの配信
3. **データ同期**: 家族間での情報共有（Cloud Firestore 経由）
4. **データエクスポート**: CSV/PDF 形式でのデータ出力
5. **アプリ改善**: クラッシュレポート、使用状況分析（匿名化）

## 第三者提供

### Firebase / Google
- **Firebase Auth**: 認証情報（メールアドレス）
- **Cloud Firestore**: 同期データ（暗号化通信）
- **Firebase Storage**: 写真データ
- **Firebase Crashlytics**: クラッシュログ（PII を含まない）
- **Firebase Analytics**: 使用状況イベント（匿名化済み）

上記サービスは Google のプライバシーポリシーに従って運営されます。

### その他の第三者
- 上記以外の第三者にデータを提供することはありません
- 広告ネットワークへのデータ提供は行いません

## データ保存場所

| 保存先 | 内容 | 暗号化 |
|--------|------|--------|
| ローカル（Room + SQLCipher） | 全アプリデータ | AES-256 暗号化 |
| ローカル（EncryptedSharedPreferences） | ユーザー設定、DB パスフレーズ | AES-256 暗号化 |
| Cloud Firestore | 同期対象データ | Google 管理の暗号化 + TLS 通信 |
| Firebase Storage | 写真データ | Google 管理の暗号化 + TLS 通信 |

## データ保持期間

| データ種別 | 保持期間 | 削除条件 |
|-----------|---------|---------|
| アプリデータ（ローカル） | アプリアンインストールまで | ユーザーによるアンインストール |
| 同期データ（Firestore） | アカウント削除まで | ユーザーによるアカウント削除 |
| 写真データ（Storage） | アカウント削除まで | ユーザーによるアカウント削除 |
| エクスポートキャッシュ | 1 時間 | 自動クリーンアップ |
| 写真キャッシュ | 7 日間 | 自動クリーンアップ |
| クラッシュログ | 90 日間（Firebase 規定） | Firebase による自動削除 |
| Analytics データ | 14 ヶ月（Firebase 規定） | Firebase による自動削除 |

## 開示・訂正・削除の手続き（APPI 第28〜34条）

### 開示請求
利用者は、アプリ内のデータエクスポート機能（CSV/PDF）を使用して、
保存されている個人情報をいつでも確認・取得できます。

### 訂正請求
利用者は、アプリ内の編集機能を使用して、
保存されている個人情報をいつでも訂正できます。

### 削除請求
以下の方法でデータを削除できます:

1. **個別データの削除**: アプリ内の削除機能で各レコードを削除
2. **アカウント削除**: 設定画面からアカウントを削除（全クラウドデータが削除）
3. **アプリアンインストール**: 全ローカルデータが削除
4. **お問い合わせ**: support-carenote@ks-apps.org にメールで削除を依頼

### 利用停止請求
利用者は、設定画面からクラウド同期を無効にすることで、
クラウドへのデータ送信を停止できます。
